AI BREAST CANCER PREDICTION SYSTEM
==================================================

PROJECT OVERVIEW
================

The AI Breast Cancer Prediction System is a Machine Learning based web application that helps predict whether a breast tumor is cancerous or non-cancerous.

The system takes medical information from the user, processes it using a trained Machine Learning model, and displays the prediction result along with a risk percentage.

The main goal of this project is to demonstrate how Machine Learning can assist in the early detection of breast cancer.

HOW TO RUN THE PROJECT
======================
Method 1: Automatic Browser Opening

1. Open Command Prompt or PowerShell.

2. Go to the project folder.
Example:

3. Activate the virtual environment (if available).
Example:

4. Install the required libraries.

5. Run the application.
python app.py

6. The browser will automatically open the project page.

HOW THE PROJECT WORKS
=====================
Step 1:
The user opens the website.

Step 2:
The user enters the breast cancer feature values.

Step 3:
The entered data is sent to the Flask backend.

Step 4:
The backend converts the input data into a format suitable for Machine Learning.

Step 5:
The trained Machine Learning model analyzes the data.

Step 6:
The model predicts whether the tumor is:

• Cancer Detected

or

• No Cancer Detected

Step 7:
The system calculates the prediction probability (risk percentage).

Step 8:
The result is displayed on the website with a prediction card and risk score.

ABOUT BREAST CANCER
===================
Breast cancer is a disease in which cells in the breast grow abnormally and uncontrollably.

There are two main types of tumors:

1. Benign Tumor



   * Non-cancerous

   * Does not spread to other parts of the body



2. Malignant Tumor



   * Cancerous

   * Can spread to nearby tissues and organs



Early detection is very important because it helps doctors begin treatment sooner and increases the chances of successful recovery.
